# APE04-Clases-Objetos-Metodos
A través del desarrollo del programa, se consiguió establecer adecuadamente un sistema básico para la gestión de estudiantes utilizando Java con programación orientada a objetos. Este sistema facilitó el registro de los datos de cinco estudiantes, abarcando su número de identificación, nombre, apellido y tres notas.

Asimismo, el programa llevó a cabo el cálculo automático del promedio de cada alumno y definió su situación académica como aprobada o reprobada en función del promedio calculado. Se validó que las calificaciones ingresadas estuvieran dentro del rango permitido de 0 a 10, lo cual ayudó a prevenir errores en la entrada de datos.
