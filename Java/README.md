El programa desarrollado en Java permite gestionar información de estudiantes aplicando programación orientada a objetos mediante clases, objetos y métodos. A través de la clase Estudiante, el sistema registra datos personales y calificaciones, calcula automáticamente el promedio y determina el estado académico de cada estudiante. Además, se implementaron validaciones de notas y se muestran los resultados finales de forma organizada y estructurada.
